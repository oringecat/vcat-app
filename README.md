# vcat-app
APP(H5+) 混合开发基础框架，可快速搭建开发环境，可打包成 Android、iOS 等应用程序，目前已实现基本操作逻辑功能，请自行体验(使用移动端模式)，后面有空的话会慢慢更新完善。

## 特性说明
- Vue3.0 + Typescript + Vant
- 基本的 APP 页面布局和用户操作逻辑
- 准确的前进后退动画，A→B→C 或 C→A
- 支持前进刷新后退缓存
- 支持调用手机原生 API
- 已封装 Http 请求接口类
- 功能拆分，组件与组件自由组合页面
- 其它 ...
